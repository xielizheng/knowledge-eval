# Changelog

本项目的重要产品与交互变更记录在此文件中。

## [v1.0.0] - 2026-07-15

- 建立知识库效果评测完整交互原型。
- 提供最终得分、`-1 / 0 / 1` 分布、待复核数量和趋势分析。
- 提供 AI 自动归因、评测流水、单条明细与页面内人工修正流程。

## [v0.5.3] - 2026-07-15

- 移除 AI 问题自动归因区域的四张汇总卡片。
- 问题归因统一为问题理解、证据相关性、证据充分性、回答忠实度、回答完整性和边界处理六个维度。
- 六个维度均采用 `0 / 1 / 2` 评分，并展示平均分、0 分数、1 分数和待改善样本数。
- 流水表以六维评分中的最低分维度作为主问题归因。

## [v0.5.2] - 2026-07-15

- 评测流水移除轮数，Query 列不再限定首轮。
- 流水结果调整为“问题归因、是否置信、得分”三列。
- 置信样本按 `-1 / 0 / 1` 输出评分；非置信样本不输出分数并标记人工复核。
- 流水工具栏调整为“导出所有”和“导出非置信”。

## [v0.5.1] - 2026-07-15

- 移除批次首页的全局人工标注导入入口，导入操作保留在具体批次详情中。
- 取消“待回流”状态，人工回流不再作为批次完成的前置条件。
- 人工回流列统一展示数字，未回流批次显示 `0`。

## [v0.5.0] - 2026-07-15

- 新增批次列表首页，点击批次进入聚合结果。
- 发起评测支持周期抽样、指定时间范围和样本规模选择。
- 移除单条在线明细与页面内人工评分，降低一期开发成本。
- 支持下载 Excel 可编辑的 CSV 校验模板。
- 支持导入人工标注 CSV 并更新批次回流状态。
- 详情指标数量与所选批次样本数联动。
